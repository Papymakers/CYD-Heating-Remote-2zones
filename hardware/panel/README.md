# Façade avant — CYD Heating Remote 2 zones

## Présentation

Façade avant pour l'installation du module CYD dans un **boîtier
d'encastrement mural** standard. Format interrupteur européen **86 × 86 mm**.

## Boîtier compatible

**Boîte d'encastrement XL AIR'métic — 2 Postes**

| Paramètre | Valeur |
|---|---|
| Diamètre | 67 mm |
| Profondeur minimale | **40 mm** |
| Fabricants | Eur'Ohm, Legrand |
| Référence | [Domomat — Eur'Ohm](https://www.domomat.com/93_eur-ohm) |

> Un pot de profondeur < 40 mm ne permet pas de loger le module CYD.
> Les pots 1 poste (Ø 60 mm) sont trop étroits pour la façade 86 × 86 mm.

## Dimensions façade

| Paramètre | Valeur |
|---|---|
| Façade | 86 × 86 mm |
| Découpe écran CYD | 76 × 57 mm |
| Trous fixation CYD | M2.5 — entraxe 70.5 × 51.5 mm |
| Trous fixation boîtier | M3.5 — entraxe 60 × 69 mm |

## Fichiers

| Fichier | Usage |
|---|---|
| `panel_front.svg` | Source vectorielle Inkscape — éditable |
| `panel_front.dxf` | Découpe laser / CNC (à générer depuis le SVG) |

### Génération DXF depuis Inkscape

1. Ouvrir `panel_front.svg`
2. Sélectionner calques `cut` et `engrave` uniquement
3. `Fichier → Enregistrer une copie → DXF R12`
4. Vérifier l'échelle 1:1 en mm

### Calques laser

| Couleur | Calque | Action |
|---|---|---|
| Rouge `#FF0000` | `cut` | Découpe traversante |
| Bleu `#0000FF` | `engrave` | Gravure texte |
| Gris `#AAAAAA` | `dim` | Cotations — ne pas graver |
